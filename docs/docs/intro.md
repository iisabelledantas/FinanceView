# FinanceView

FinanceView é uma aplicação mobile de finanças pessoais com backend serverless na AWS. O projeto permite autenticar usuários, importar extratos bancários em PDF, revisar transações, categorizar gastos automaticamente, acompanhar receitas/despesas, criar metas de orçamento, analisar indicadores financeiros e compartilhar relatórios.

Esta documentação descreve o projeto de ponta a ponta: app Flutter, Lambdas Python, infraestrutura Terraform, banco de dados, fluxos principais, execução local, deploy, testes, problemas encontrados e próximos passos.
